Visual Memorize

Visual Memorize helps you to learn associations with images. For every PDF you drop here the following happens:
  1. Visual Memorize shows you the first picture on a random page of your PDF
  2. You press the "arrow right" key, click or tap the image to see the full page of the PDF
  3. One you press the "arrow right" key, click or tap the PDF page again, the first image of another random page is shown and the process repeats.
Drop a file anywhere in this window or select a file in the dialog
